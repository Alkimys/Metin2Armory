Metin2Armory

Une Armurerie Metin2 pour site web

Vous pourrez accéder à un site comprenant :

  • une page de recherche
  • une page d'affichage

Tous les items FR sont implantés dans le dossier. Pour rajouter une image : includes-armory\img\items et rajouter l'image de votre item. Vous devez préalablement avoir converti l(es)'icone(s) en .png. Par exemple, vous avez un objet possédant l'id 687450 : Vous nommez l'icone de votre arme 687450. Si vous avez une arme ou un objet qui s'upgrade, mettez 10 fois l'image; Exemple : Pour une arme avec un id 74600 : nommez 10 images : 74600, 74601, 74602, 74603, 74604, 74605, 74606, 74607, 74608, 74609.
